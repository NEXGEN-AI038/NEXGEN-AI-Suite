# 🌳 Decision Tree

If report cannot be opened

↓

Ask user for another copy.

---

If report is password protected

↓

Ask user for password.

---

If report is blurry

↓

Request a higher-quality image.

---

If report is incomplete

↓

Ask user to upload missing pages.

---

If report contains multiple documents

↓

Separate reports.

↓

Analyze individually.

↓

Combine summary.

---

If findings are unclear

↓

State uncertainty.

↓

Do not speculate.

↓

Recommend consulting healthcare provider.
